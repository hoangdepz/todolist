App Setup

1. Download and Install Android Studio

Link: Android Studio

Import Project

  1. Download the project zip file.
  2. Unzip the file to find the project in the "Code" folder.
  3. Open Android Studio and select "Open an existing Android Studio Project".
  4. Browse to the project location and click "OK".
  5. Wait for the import process to complete.

Change Package Name

AdMob Ad Setup

  1. Log in to your AdMob account, create an app, and generate your ad unit ID.
  2. Update the IDs in res/values/strings.xml:
<!-- AdMob IDs -->
<string name="admob_app_id">YOUR_ADMOB_APP_ID</string>
<string name="admob_banner_ad">YOUR_ADMOB_BANNER_ID</string>

Change App Name and Other Strings

  1. Open res/values/strings.xml.
  2. Modify the app name and other string resources:
<string name="app_name">Your App Name</string>

Change App Logo

  1. Navigate to the res folder.
  2. Replace the launcher icon in the mipmap or drawable folder.

Change App Color

Open res/values/colors.xml and modify the color values as needed.

Generate Signed APK

  1. Click Build > Generate Signed Bundle / APK > APK.
  2. If you donâ€™t have a keystore, create a new one.
  3. Provide the required information for the keystore.
  4. Select the keystore, private key, and enter the passwords.
  5. Choose a destination for the signed APK.
  6. Select Signature Version: V1 (Jar Signature).
  7. Click Finish to generate the signed APK.

Change Ad Units

  1. Open the project in Android Studio.
  2. Locate AdsGoogle.java in the main package.
  3. Replace the placeholder ad IDs with your real IDs:
String AD_Banner_ID = "YOUR_AD_BANNER_ID";
String AD_Interstitial_ID = "YOUR_AD_INTERSTITIAL_ID";
